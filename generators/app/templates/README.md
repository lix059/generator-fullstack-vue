## Build Setup

``` bash
# install dependencies
npm install

    npm run dev

    # build for production with minification
    npm run build

    # run unit tests
    npm run unit

    # run e2e tests
    npm run e2e

BE: 
    # serve with hot reload at localhost:9000
    grunt server

    # build for production, when package project must be run after FE build
    grunt build

    grunt test
